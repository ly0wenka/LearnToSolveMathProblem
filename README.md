# MathMentor Flow

`MathMentor Flow` is a lightweight educational web application created as a concept for the bachelor thesis topic:

`Проєктування та розробка програмного забезпечення для навчання з розв'язання математичних задач`

## What is included

- adaptive one-page interface for presenting the diploma project idea;
- learning modules with outcomes and competency focus;
- interactive problem workspace with step-by-step solving guidance;
- self-check form with local progress saving in `localStorage`;
- architecture section that can be reused in thesis documentation.

## Run locally

Open `index.html` in a browser.

For a local server in PowerShell:

```powershell
php -S localhost:8080
```

or

```powershell
python -m http.server 8080
```

Then open `http://localhost:8080`.
