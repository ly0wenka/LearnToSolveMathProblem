# Project Rules

## Topic

Bachelor thesis software topic:

`Проєктування та розробка програмного забезпечення для навчання з розв'язання математичних задач`

## Product focus

- Web application for guided learning of mathematics.
- Target audience: pupils, students, and tutors.
- Core value: step-by-step explanation instead of final-answer-only checking.
- Main entities: learning module, problem, solving step, learner progress, recommendation.

## Functional accents

- Present theory in short structured blocks.
- Support practical tasks with incremental hints.
- Check answers and store local progress.
- Show architecture ideas suitable for thesis diagrams and explanatory note.
